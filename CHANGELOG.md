# Changelog

All notable changes are documented here. The project follows Semantic Versioning.

## [0.1.0] - 2026-07-21

### Added

- Technitium DNS Server 15.4 query logger integration.
- Non-blocking bounded queue and background UDP exporter.
- dnsmasq-compatible RFC3164 formatting for UniFi Insights Plus 3.7.0.
- Strict configuration, node allowlist, filters, counters, rate-limited errors, and bounded graceful shutdown.
- Unit, UDP integration, API compatibility, concurrency, shutdown, and load tests.
- Deterministic package build, SHA256 generation, CI, CodeQL, migration, security, and rollback documentation.
